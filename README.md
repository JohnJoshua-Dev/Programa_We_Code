**Aula 1 Git**
git init ->Para Iniciar/criar um repositorio 
git add . ->Adiciona os ficheiros para serem commitados
git add nomedoFicheiro -> Adiciona um ficheiro especifico para ser commitado
git commit -> Escreve a mudança (git commit -m"Comentario")
git commit -a -m"Comentario"

git clone url -> Clona o repositorio remoto para ambiente local

git pull "remoto" "branch" (git pull origin main) -> Pusha do remoto para meu repositorio local 

git push "remoto" "branch" (git push origin main) -> sobe as alterações que tenho no repositorio local para o remoto

git remote add "remoto" url

exemplo clone: git clone https://github.com/JohnJoshua-Dev/Programa_We_Code.git
