### Arrays, if else e Estruras de repetição
Estruras de Repetição:
for -> Repete um numero de vezes conhecidos 
Estrutura
```text
    for(int i=0; i<5; i++){
        codigo a ser repetido 5 vezes
    }
```
i++ = i+1 (encremento)
i-- = i-1 (decremento)

forEach -> Percore todos os dados de um array
```text
    for(String nome : nomes){
        system.out.print(nome)
    }
```

While / do while -> Estrura de repetição quando não conhecemos os limites ou nr de vezes a repetir
While verifica acondição antes de executar, do while executa e depois verifica

Estrutura
while
```text
    while(condição){
        codigo a repetir
    }
```
do while
```text
    {
        codigo a repetir
    }
    while(condição)
```

If Else -> Estrutura se coloca condições para alguma coisa acontecer
//Condição
```text
if(condição a verificar){
    codigo a fazer se for verdade
}

//if else
if(condição a verificar){
    codigo a fazer se for verdade
}else {
    codigo a ser feito caso nao seja verdade
}
```
